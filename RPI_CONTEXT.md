# RPI Context

## Tech Stack
- Frontend: React + Tailwind
- Backend: Django
- Base de datos: Por definir (por ahora sin base de datos)

## Dominio de Negocio

El sistema se basa en tres entidades fundamentales:

1. **Folios**: Documentos, expedientes o registros numéricos del sistema
2. **Trámites**: Procesos administrativos gestionados sobre los folios
3. **Asientos**: Movimientos o registros contables asociados a los trámites

## Arquitectura
- Backend Django como API REST
- Frontend React con componentes Tailwind
- Sin base de datos actual
